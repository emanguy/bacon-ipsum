bacon-ipsum
===========

A meatier Lorem Ipsum generator provided via a Polymer custom element.

Install
-------

```
bower install bacon-ipsum
```

Example
-------

```
<bacon-ipsum paragraphs="3"></bacon-ipsum>
```
